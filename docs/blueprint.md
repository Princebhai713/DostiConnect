# **App Name**: DostiConnect

## Core Features:

- User Authentication: Secure user registration and login with username and password.
- Friend Request System: Users can send friend requests using User IDs. Users can accept/decline friend requests.
- Chat Messaging: Real-time text messaging with friends.
- Audio Calling: Initiate and receive audio calls with friends.
- Video Calling: Initiate and receive video calls with friends.
- Friend Suggestions: AI powered tool which shows suggestions to users for new connections.

## Style Guidelines:

- Primary color: Soft blue (#7EC8E3) for a calming and friendly feel.
- Background color: Light gray (#F0F0F0) to provide a clean and modern look.
- Accent color: Green (#A7D1AB) for interactive elements and notifications.
- Body and headline font: 'PT Sans', a modern yet humanist sans-serif for both headlines and body text.
- Use flat, minimalist icons for navigation and actions.
- Clean and intuitive layout with easy navigation between chats, friend lists, and profile settings.
- Subtle animations for sending/receiving messages and friend requests.
